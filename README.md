BFS
===

Вспомогательный алгоритм (поиск в ширину) для реализации микро-игры Game1.
Суть алгоритма найти кратчайший путь в лабиринте до определенной точки из начальной/стартовой.

Я вынес реализацию алгоритма в отдельный проект т.к. думаю, что придется еще не раз к нему обращаться.
Намного проще иметь реализацию алгоритма в "чистом" виде, т.к. в каждом случае его применения, алгоритм
"дотачивается" под задачи проекта.
